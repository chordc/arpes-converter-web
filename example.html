Example Code Cells

Each section corresponds to a typical notebook step and supports one-click copying.

Cell 1 · Load the NPZ file

NPZ

Import NumPy and open the exported NPZ container. 

import numpy as np
import json

dat = np.load("ibw_to_npz.npz", allow_pickle=False)

Cell 2 · Show available keys

NPZ

Inspect what arrays and metadata entries are stored in the file. 

print(dat.files)

Expected output

['data', 'axis_0', 'axis_1', 'meta_json']

Cell 3 · Check the main data shape

NPZ

Verify the dimensions of the ARPES intensity matrix before plotting. 

print(dat["data"].shape)

Cell 4 · Inspect axis arrays

NPZ

Check axis length first, then inspect actual axis values. 

print(dat["axis_0"].shape)
print(dat["axis_1"])

Cell 5 · Plot the data

NPZ

Use the exported axes to define the image extent in a notebook plot. 

import matplotlib.pyplot as plt

plt.imshow(
    dat["data"],
    aspect="auto",
    origin="lower",
    extent=[
        dat["axis_1"][0], dat["axis_1"][-1],
        dat["axis_0"][0], dat["axis_0"][-1]
    ]
)
plt.xlabel("Axis 1")
plt.ylabel("Axis 0")
plt.title("ARPES map from NPZ")
plt.colorbar()
plt.show()

Cell 6 · Print metadata JSON

NPZ

Read the metadata payload written during conversion. 

meta = json.loads(dat["meta_json"].item())
print(meta)

Cell 7 · Load the .nc file

NetCDF

Load the converted NetCDF file with xarray. If needed, set the engine explicitly. 

import xarray as xr

da = xr.open_dataarray("npz_to_xarray.nc", engine="scipy")

Cell 8 · Inspect the xarray object

NetCDF

Print the loaded DataArray and inspect dimensions, coordinates, and attributes. 

print(da)
print(da.dims)
print(list(da.coords))
print(da.attrs)

Cell 9 · Plot the NetCDF data

NetCDF

Plot the DataArray directly with xarray. 

da.plot()

Cell 10 · Parse JSON-like attrs

NetCDF

Some metadata fields may be stored as JSON strings in NetCDF attrs. Parse them when needed. 

import json

experiment = da.attrs.get("experiment", "")
if experiment:
    experiment = json.loads(experiment)

print(experiment)
